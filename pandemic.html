Pandemic is a cooperative game designed by Matt Leacock for Zman games. This page is a solitaire implementation that contains a crude map which logically matches the map used in the actual game. For a more aesthetically pleasing and multiplayer version of this game, try the Brettspielwelt website. Click here for a page of instructions on how to play this version.
